# DC-DC-Buck-Converter-Design-and-Performance-Analysis

This project covers the design, simulation, and experimental verification of a DC-DC Buck Converter circuit aimed at step-down voltage regulation.

## 📋 Project Overview
The Buck Converter is designed to step down a higher input DC voltage to a lower regulated output DC voltage. This project includes theoretical component sizing, circuit schematic design, and experimental performance validation.

## 🛠 Circuit Specifications
* **Input Voltage:** 5V
* **Output Voltage:** 3.3V
* **Key Components:** MOSFET/IGBT, Inductor (L), Capacitor (C), Diode, PWM Controller (SG3525/Arduino/etc.)

## 🚀 Experimental Validation
The performance was verified through laboratory testing:
* **Test Case A:** Voltage regulation under no-load conditions.
* **Test Case B:** Transient response during load step changes.
* **Test Case C:** Efficiency and ripple analysis observed via oscilloscope.
